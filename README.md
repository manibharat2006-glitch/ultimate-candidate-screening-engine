# Ultimate Candidate Screening Engine

A **Python-based HR tool** for **automated candidate screening**.  

- GUI built with **Tkinter**  
- Advanced **skills scoring engine**  
- Weighted scoring of **core, soft, optional skills**  
- **Experience & keyword analysis**  
- **Strength highlights and suggestions**  
- **Color-coded professional dashboard**  

## Features

- Paste resume text or load from `.txt` file  
- Paste job description or load from `.txt` file  
- Run advanced scoring and generate **final decision**: Shortlisted / Needs Review / Rejected  
- Save screening results to a file  
- Loop for screening **multiple candidates**
